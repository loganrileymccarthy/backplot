# Lane4 Backplotter
May 2025  
Now a react js app running in browser!  

[Click here to launch site](https://loganrileymccarthy.github.io/backplot)  

Features:
+ Coordinates, current tool, g/m codes for each line
+ Z/X, Z/Y, X/Y views (spindle 1)
